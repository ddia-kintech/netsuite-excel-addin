NetSuite Excel Add-in

This is the homepage for the NetSuite Excel Add-in. This add-in allows you to integrate Excel with NetSuite for efficient financial data management.

Install the Add-in

To use this add-in in Excel:

  1. Download the manifest file
  2. Open Excel
  3. Go to Insert tab > My Add-ins > Upload My Add-in
  4. Select the downloaded manifest file
Download manifest.xml

Features

  • Import journals, bills, invoices, and purchase orders from Excel to NetSuite
  • Download NetSuite data (accounts, subsidiaries, etc.) to Excel
  • Support for custom forms and fields
  • Secure Token-Based Authentication
Note: This add-in requires valid NetSuite credentials with appropriate permissions.
